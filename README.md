# ⚛️ Quantum Principal Component Analysis (QPCA) Web App

This is a **Streamlit-based web application** that demonstrates **Quantum Principal Component Analysis (QPCA)** using classical simulation with density matrices. The app allows you to visualize the density matrix, principal components, scree plot, and Bloch sphere representations interactively.

---

## **Features**

- ✅ Use a demo dataset or upload your own CSV dataset (numeric features only).  
- ✅ Step-by-step computation of the **density matrix**.  
- ✅ Visualization of the **density matrix** using a histogram.  
- ✅ Computation and display of **eigenvalues** and **principal components**.  
- ✅ Interactive **Bloch sphere** visualization of principal components.  
- ✅ Scree plot to analyze the variance explained by each component.  
- ✅ Export PCA results to **CSV** or **LaTeX** format.
